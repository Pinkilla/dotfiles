#dotfiles

Some of my configuration files. 

## list
### vim
*vim* dotfile. 

To install it

* copy `vim/vimrc` to `~/.vimrc`;
* create `~/.vim` rep;
* add [Pathogen][pathogen] plugin and eventually [NERDTree][nerdtree];

See [Steve Losh post][slp] and my post on [Howto vim][notespit-vim].
